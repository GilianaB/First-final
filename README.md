1. Задаем длину массива.
2. Объявляем первый массив строк.
3. Объявляем второй массив строк.
4. Задаем цикл и заполняем первый строковый массив.
5. Объявляем переменную count и приравниваем ее к нулю. Данная переменная будет счетчиком для длины нового массива.
6. Задаем цикл для перебора всех данных первого массива.
7. Задаем операцию сравнения длины каждого элемента первого массива , менее или равно 3.
8. Если длина элемента первого массива менее или равна 3, то данный элемент записывается в новый массив. Счетчик увеличивается на одну позицию.
9. Операция продолжается до окончания перебора всех элементов массива.
10. Задаем цикл и выводим на печать новый массив строк.
